安装

在Steam中打开Cuphead的文件夹

打开该目录下的Cuphead_Data/Managed/文件夹
将该目录下的Assembly-CSharp.dll替换为CupheadLeziMod仓库的同名文件
打开游戏，进入岛屿，按下F1，如果出现以下界面则安装成功
