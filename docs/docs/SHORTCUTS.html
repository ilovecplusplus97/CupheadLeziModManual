快捷键

快捷键

F1, F2, F5, F6

F1为打开或关闭DEBUG菜单
F2为切换DEBUG菜单界面
F5为开启或关闭垂直同步
F6为卸下护符和大招

1, 2, 3, 5, 6, 8, 9, 0

1为增加一格能量
2为填满所有能量
3为显示或关闭帧数
5为减少一个金币
6为减少一个金币
8为提高大量伤害
9为使角色无敌
0(已废弃)为打开选关界面

其他

LeftCtrl + Keypad+为提高游戏速度
LeftCtrl + Keypad-为降低游戏速度
LeftCtrl + Keypad0为恢复游戏速度为正常
LeftCtrl + PageUp为放大游戏画面
LeftCtrl + PageDown为缩小游戏画面
LeftCtrl + End为恢复游戏画面为正常
R为关卡重来
